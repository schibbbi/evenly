# AI Involvement

Evenly was built with significant assistance from [Claude](https://www.anthropic.com/claude) by Anthropic,
used as an AI coding agent throughout the entire development process.

## What was AI-assisted

- Architecture and technology decisions
- All backend code (FastAPI, SQLAlchemy models, Alembic migrations, business logic agents)
- All frontend code (Vue.js 3, Vuetify 3, vue-i18n, API client)
- Database schema design and migration scripts
- Docker / Podman Compose configuration
- Documentation (README, installation guides)
- Debugging and bug fixes

## What was human-driven

- Product idea and concept
- Feature definitions and requirements per development round
- UX decisions and wording
- Final review and deployment

## Tools used

| Tool | Purpose |

| [Claude](https://www.anthropic.com/claude) by Anthropic | AI coding agent (primary) |

## Note on transparency

This project is shared publicly as-is. The AI-assisted nature of the code does not affect
its license or usability. All code has been reviewed by the project owner before publishing.
